# Maya's 3rd Birthday 🎂

An interactive, mobile-first birthday invitation built as a single self-contained
`index.html` (no build step, no dependencies). Designed to be shared as a WhatsApp
link and opened on phones.

**Live:** https://eitan-shteinberg.github.io/maya-bday/

## Details
- **Who:** Maya — turning 3 🎉
- **When:** Wednesday, July 29, 2026 · 4:00 PM
- **Where:** Lake Hills Park, 164th Ave SE & SE 12th St, Bellevue, WA 98008

## Features
- 🎁 Tap-to-open reveal with confetti
- 🎈 Animated hero with floating balloons
- 🎂 Interactive "blow out the candles" cake (tap or blow into the mic)
- ⏳ Live countdown to the party
- 📍 Embedded map + one-tap Google Maps directions
- 🗓️ Add to Calendar (Google + Apple/Outlook `.ics`)
- 🔗 Share button + rich WhatsApp link preview

## Editing
Everything lives in `index.html`. To change the date/time, update the countdown
target and the calendar block (search for `2026-07-29`). `preview.png` is the
social/link-preview image.
